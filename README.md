# To-Do List Project

This is a simple To-Do List web application that allows users to add, edit, and delete tasks. The tasks are stored in the browser's local storage, ensuring that they persist even after the page is refreshed.

## Features

- Add new tasks to the list

- Edit existing tasks

- Delete tasks from the list

- Tasks are stored in local storage

- Alternating background colors for tasks for better readability

## Technologies Used

- HTML

- CSS

- JavaScript

## Installation

- Clone the repository or download the source files.

- Open the index.html file in your browser.
